# B-전략 그룹 Deep Research 가이드

> 3교시 실무 반복 루프에서 내부 시장 자료만으로 부족할 때 사용하는 외부 리서치 가이드입니다.
> 이 문서는 VSCode extension 채팅창의 `/deep-research` 슬래시 커맨드 사용을 전제로 합니다.
> 경쟁사, 정책, 시장 구조 변화는 항상 공개 정보 수준에서만 다룹니다.

## 언제 쓰나요?

- `practice-data/B-전략/` 내부 자료만으로 경쟁사 맥락이 부족할 때
- 최신 경쟁사 IR, 산업 정책, 공급망 재편, 점유율 변화가 궁금할 때
- 분기별 시장 보고서에 외부 근거를 한두 개 추가하고 싶을 때
- 전략팀 발표 자료에 "최근 공개 동향" 한 장이 필요할 때
- Q1, Q2 시장 보고서의 서술을 최신 공개 자료와 연결해 보고 싶을 때

## 먼저 확인할 내부 파일

- `practice-data/B-전략/market_reports/Q1-2026.md`
- `practice-data/B-전략/market_reports/Q2-2026.md`
- `practice-data/B-전략/competitor_ir_2026Q1.md`

## VSCode extension에서 /deep-research 호출하기

VSCode 채팅창에서 아래처럼 바로 입력합니다.

```text
/deep-research 경쟁사 최신 IR 리포트 핵심만 요약해줘
```

또는 이렇게 더 분명하게 적어도 됩니다.

```text
superpowers:deep-research 스킬로 북미 IRA 최근 변경 사항을 조사해줘
```

비교 형식이 필요하면 이렇게 요청합니다.

```text
/deep-research 현대차그룹, 포스코퓨처엠, LG에너지솔루션의 최근 공개 자료를 비교해서 투자 포인트와 리스크를 표로 정리해줘
```

## 전략팀 권장 쿼리 예시

1. `/deep-research 경쟁사 최신 IR 리포트 요약`
2. `/deep-research 자동차 부품 산업 M&A 동향`
3. `/deep-research 중국 완성차 시장 점유율 변화`
4. `/deep-research 북미 IRA 최근 변경 사항`
5. `/deep-research 전고체 배터리 공급망 재편 동향`
6. `/deep-research 한국 자동차 부품사 북미 투자 확대 사례`
7. `/deep-research 최근 6개월 환율 변화가 자동차 부품 수익성에 미친 영향`

## 추천 입력 템플릿

```text
/deep-research {주제}를 조사해줘.
대상은 자동차 부품사 전략팀이고,
최근성은 최근 6개월 우선,
출처 URL과 발표일을 함께 적어주고,
시장 영향, 경쟁사 시사점, 우리 대응 포인트를 나눠서 정리해줘.
```

## 결과를 기존 분석에 합치는 법

1. `/deep-research` 결과를 `외부리서치_{날짜}.md`로 먼저 저장합니다.
2. `Q1-2026.md` 또는 `Q2-2026.md`를 읽히면서 같이 참고하도록 전달합니다.
3. 예: "`Q2-2026.md` 경쟁사 동향 문단을 `외부리서치_2026-04-06.md`의 최신 공개자료와 연결해서 다시 써줘"
4. `competitor_ir_2026Q1.md`와 함께 비교하면 분기 보고서 톤으로 재정리하기 쉽습니다.

## 추천 작업 순서

1. 내부 보고서에서 부족한 문장을 표시합니다.
2. 부족한 문장을 질문으로 바꿉니다.
3. VSCode 채팅창에서 `/deep-research`를 실행합니다.
4. 출처와 날짜를 먼저 확인합니다.
5. 경쟁사, 정책, 시장 자료를 구분합니다.
6. 우리 전략에 직접 연결되는 문장만 남깁니다.
7. 보고서 한 문장으로 줄입니다.

## 시장 보고서와 연결하는 질문 예시

- Q2 보고서의 "공급 안정성 중시" 흐름은 실제 최근 발표 자료와 맞나요?
- 경쟁사들의 AI 검사 홍보는 어느 정도까지 공개 자료에서 확인되나요?
- 북미 생산거점 보유 여부가 최근 RFQ나 투자 발표에서 얼마나 강조되나요?
- ESG 데이터 요구가 실제 공개 문서에서도 강해지고 있나요?

## competitor_ir와 연결하는 질문 예시

- 3사 공통 키워드를 최근 공개 자료로 다시 확인해줄 수 있나요?
- 가상 IR 요약의 투자 방향과 비슷한 실제 공개 트렌드가 있나요?
- 전고체, 북미 투자, 중국 리스크 중 가장 자주 언급되는 항목은 무엇인가요?
- 공급망 재편이 부품사에 주는 압박은 어떤 형태로 나타나나요?

## 좋은 질문과 아쉬운 질문

좋은 질문 예시:

- `/deep-research 최근 6개월 자동차 부품 산업 M&A 동향을 거래 목적 중심으로 정리해줘`
- `/deep-research 중국 EV 시장 점유율 변화가 한국 부품사 전략에 주는 시사점을 설명해줘`
- `/deep-research 경쟁사 최신 IR에서 북미 투자와 전고체 관련 언급만 뽑아 비교해줘`

아쉬운 질문 예시:

- `/deep-research 시장 알려줘`
- `/deep-research 경쟁사 전부 정리`
- `/deep-research 정책 뭐 바뀜`

## 출처 기록 방식

```markdown
## 참고 출처
- 제목:
- 기업 또는 기관:
- 발표일:
- URL:
- 전략상 의미:
```

## 실습에 맞는 정리 포맷

```markdown
## 외부 리서치 요약
- 조사 주제:
- 확인 날짜:
- 핵심 발견 1:
- 핵심 발견 2:
- 전략 시사점:
```

## 검증 체크리스트

- 날짜가 적혀 있는가
- URL이 있는가
- 기업 자료와 언론 자료가 구분되는가
- 우리 전략에 연결되는 문장이 있는가
- 경쟁사 발표를 과장 없이 옮겼는가
- 숫자 단위가 명확한가

## 자주 쓰는 주제 묶음

- 경쟁사 IR
- 북미 IRA
- 중국 완성차 시장
- 전고체 공급망
- 자동차 부품 산업 M&A
- 환율과 수익성

## 실무 연결 문장 예시

- "최근 공개 자료를 보면 북미 현지 생산과 공급 안정성 강조가 계속 이어지고 있어, 전략팀은 단가 경쟁만 보지 않는 편이 좋습니다."
- "전고체 배터리 파일럿 언급은 여전히 많지만, 단기 수익보다 중장기 포지셔닝 성격이 강하다고 볼 수 있습니다."
- "중국 시장 리스크는 가격 경쟁과 수요 변동을 동시에 봐야 하므로 단일 지표로 해석하지 않는 편이 좋습니다."

## 해석 시 주의할 점

- 언론 기사와 공식 IR 자료는 무게가 다릅니다.
- 특정 기업의 투자 발표는 실제 집행과 차이가 있을 수 있습니다.
- 공개 수치는 연결 기준, 지역 기준, 사업 기준이 다를 수 있습니다.
- 경쟁사 홍보성 표현은 실적과 분리해서 봅니다.
- 한 개 분기만 보고 구조적 추세로 단정하지 않습니다.

## 짧은 예시 시나리오

- 북미 현지 생산과 인증 대응 속도 언급이 반복되는지 확인합니다.
- 보고서 시사점 문장을 조금 더 구체화합니다.

- 전고체 파일럿, 북미 투자, 중국 리스크가 반복되는지 확인합니다.

- 가격 경쟁, 로컬 업체 약진, 수익성 압박을 분리해 정리합니다.
- 영업 전략과 공급망 대응 포인트를 연결합니다.

## 주의사항

- `/deep-research`는 VSCode extension 채팅창에서 사용하는 기능입니다.
- 강의장 네트워크 상태에 따라 실행이 어려울 수 있습니다.
- 외부 데이터는 반드시 **출처(URL + 날짜)** 와 함께 기록합니다.
- AI 요약은 초안이며 최종 판단은 수강생이 직접 합니다.
- 경쟁사 관련 내용은 공개 정보 수준에서만 다룹니다.

## 전략팀 한정 팁

- 전략팀은 내부 보고서를 먼저 읽고 외부 자료는 빈칸을 채우는 용도로 쓰는 편이 좋습니다.
- 한 번에 모든 경쟁사를 파고들기보다 의사결정에 바로 연결되는 한 주제를 먼저 보는 편이 효율적입니다.
- 분기 보고서는 공통 키워드 3개와 리스크 3개만 잡아도 충분히 실무적입니다.
